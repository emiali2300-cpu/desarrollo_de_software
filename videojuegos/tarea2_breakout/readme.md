Para correr el juego es necesario que se habra el archivo "breakout.html" para minimisar la probabilidad de algun problema es preferible que se abra con el live serve(abajo a la izquierda dentro de VS code hay un texto que dice go live, ese es)


los controles del juego son:

"A", para mover la paleta a la derecha
"D", para mover la paleta a la izquierda
" Espacio", para iniciar el juego o para volver a empezar una ves pierdes
"G", para aumentar la velocidad de la vola

El objetivo es destruir todos los bloques antes de que se acabe el tiempo,+.

La modificaciones que añadi fueron:

Cada 5 bloques rotos aumenta el tiempo
cada 10 bloques rotos se agrgan 2 vidas al jugador
si el tiempo se acaba el jugador pierde
Se puede aumentar la velocidad del juego
